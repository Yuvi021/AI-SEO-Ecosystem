# Open Graph Images Guide

This folder should contain the following Open Graph images for social media sharing:

## Required Images

### 1. og-image.png (Main/Home Page)
- **Size:** 1200 x 630 pixels
- **Content:** AI SEO Ecosystem logo, tagline "Multi-Agent SEO Analysis Platform"
- **Background:** Gradient (cyan to blue to purple)
- **Text:** "Harness the power of 9 AI agents"

### 2. og-analyze.png (SEO Analysis Page)
- **Size:** 1200 x 630 pixels
- **Content:** Dashboard/analytics visualization
- **Text:** "AI-Powered Website SEO Analysis"
- **Icons:** Charts, graphs, checkmarks

### 3. og-keyword.png (Keyword Research Page)
- **Size:** 1200 x 630 pixels
- **Content:** Keyword bubbles or word cloud
- **Text:** "AI Keyword Research & Analysis"
- **Icons:** Magnifying glass, keywords, trending arrows

### 4. og-blog.png (Blog Generator Page)
- **Size:** 1200 x 630 pixels
- **Content:** Document/writing visualization
- **Text:** "AI Blog Content Generator"
- **Icons:** Pen, document, AI sparkles

## Design Guidelines

- **Format:** PNG (supports transparency)
- **File size:** Keep under 1MB for fast loading
- **Safe zone:** Keep important text/logos 40px from edges
- **Font size:** Minimum 60px for readability
- **Contrast:** Ensure text is readable on all backgrounds
- **Brand colors:** 
  - Cyan: #06b6d4
  - Blue: #2563eb
  - Purple: #9333ea

## Tools for Creating OG Images

1. **Canva** - https://www.canva.com (Free templates available)
2. **Figma** - https://www.figma.com (Professional design tool)
3. **OG Image Generator** - https://og-image.vercel.app (Quick generation)
4. **Photopea** - https://www.photopea.com (Free Photoshop alternative)

## Testing Your OG Images

After creating images, test them with:
- **Facebook Debugger:** https://developers.facebook.com/tools/debug/
- **Twitter Card Validator:** https://cards-dev.twitter.com/validator
- **LinkedIn Post Inspector:** https://www.linkedin.com/post-inspector/
