# Extension Icons

## Quick Setup

You have two options to create the required icon files:

### Option 1: Use the Icon Generator (Easiest)

1. Open `create-icons.html` in your browser (double-click the file)
2. Click "Download All Icons"
3. The icons will download automatically
4. Move them to this `icons/` folder

### Option 2: Create Icons Manually

Create three PNG files:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

You can use:
- Online icon generators (search "favicon generator")
- Image editors (Photoshop, GIMP, Canva)
- AI image generators

### Option 3: Use Simple Colored Squares (Temporary)

If you just want to test the extension quickly, you can create simple colored square images using any image editor or online tool.

## Icon Requirements

- Format: PNG
- Background: Transparent or solid color
- Recommended: Simple design with "SEO" text or a magnifying glass icon
- Colors: Use the brand colors (#667eea to #764ba2 gradient) if possible

1. Place all three icon files in this `icons/` folder

3. The extension should now load without errors!
